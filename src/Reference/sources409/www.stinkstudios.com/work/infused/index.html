Screen Tests

AT&T Infuse

Screens so vivid, they confound nature.

See the project

See how real animals react to the brilliant color display of AT&T and Samsung's new Infuse 4G phone. In this campaign, which featured a Facebook app and a number of different rich media banner executions, users could select any of six doors and watch various animals interact with nature via the Infuse's vivid new screens.

This might have been the best shoot ever. Over the course of one day, along with our director Mason Nicoll, we had the privilege of working with the most exotic of talents: an owl, a frog, a fish, a mouse and of course, the star of the show, Tania, our kitten.
