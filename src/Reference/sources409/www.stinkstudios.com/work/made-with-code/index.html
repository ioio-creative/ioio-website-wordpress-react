Made With Code

Google

A unique resource dedicated to helping young women get involved with code and creative technology.

See the project

Stinkdigital partnered with 72andSunny to redesign madewithcode.com to create an online resource for young girls who are interested in coding. 74% of girls express interest in Science, Technology, Engineering, and Math (STEM) in middle school. By high school, only 0.3% girls plan to major in Computer Science. The aim of Made with Code is to provide a place for girls to visit, explore, and learn the principles of coding.

Made with Code is fully responsive, with design and animations inspired by Google's Material Design. Stinkdigital built a python based CMS to manage site content, allowing for easy maintenance and updates as the Made with Code community grows and evolves.

Made with Code features videos of inspiring female coders and how they apply their trade in the real world, images and videos from the coding community, access to coding events around the country, and online projects girls can use to learn the basics of coding principles.

Made with Code promotes the growth of the coder community by allowing users to submit local coding events to the Events page. We also partnered with Code.org to aggregate the most complete list of events across the country

Utilizing Block.ly, one of the key features are the custom games created for Made with Code that outline coding principles like conditional events, looping events, variables, functions, and sequences.
